### Prerequisites

- **Go** 1.21+
- **Rust** 1.70+
- **Python** 3.11+
- **PostgreSQL** 14+ (for explorer)
- **Docker** 20+ (optional)

### Installation

```bash
# Clone repository
git clone https://github.com/alejandrozahran-cyber/zahran-2-chain.git
cd zahran-2-chain

# Build core blockchain
cd l1-core
go build -o nusa-node ./cmd/node

# Build VM layer
cd ../l2-vm
cargo build --release

# Setup AI layer
cd ../l3-ai
pip install -r requirements.txt

# Run node
./l1-core/nusa-node --config config.yaml
```

    use std::string;

        text: string::String
    }

        });

}
